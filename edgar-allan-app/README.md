# Edgar Allan EOY

A multi-step interactive LEGO building experience with two paths: ordering LEGO sets and guided building with personalized questions.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```
